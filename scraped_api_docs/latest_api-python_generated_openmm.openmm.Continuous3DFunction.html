Create a Continuous3DFunction f(x,y,z) based on a set of tabulated values.

Parameters

  • values (int) â the tabulated values of the function f(x,y,z) at xsize uniformly spaced values of x between xmin and xmax, ysize values of y between ymin and ymax, and zsize values of z between zmin and zmax. A natural cubic spline is used to interpolate between the tabulated values. The function is assumed to be zero when x, y, or z is outside its specified range. The values should be ordered so that values[i+xsize*j+xsize*ysize*k] = f(x_i,y_j,z_k), where x_i is the iâth uniformly spaced value of x. This must be of length xsize*ysize*zsize.
